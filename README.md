# rmaruy3.github.io
 
丸山隆一の個人ウェブサイトです。
 
🌐 **https://rmaruy3.github.io**
 
---
 
## 概要
 
ブログ記事・ノート・ツイートアーカイブを公開しています。
 
| セクション | 内容 |
|---|---|
| `blog/` | ブログ記事 |
| `note/` | メモ・ノート |
| `tweets/` | ツイートアーカイブ |
 
---
 
## 技術スタック
 
- **静的サイトジェネレーター**: [Jekyll](https://jekyllrb.com/)
- **ホスティング**: [GitHub Pages](https://pages.github.com/)
- **CI/CD**: GitHub Actions (`.github/workflows/`)
- **検索**: `search.json` ベースのクライアントサイド検索
